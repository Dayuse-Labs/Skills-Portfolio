# Dayuse Icon Catalog

Source: Figma Design System v2.0.3
File: https://www.figma.com/design/1kQ21QwHaQxQrfR1Ozz7ys/Sunrise---Dayuse.com-Design-system-v2.0.3?node-id=40-927

## Overview

187 custom icons across 8 categories. All icons are outline/line style in Evening Blue (#292935).

### Exported SVGs — File Mapping

Icons have been exported to `assets/icons/` in two folders:

| Folder | Prefix | Count | Categories Covered |
|--------|--------|-------|--------------------|
| `UI_icon/` | `nav-` | 20 | Navigation |
| `UI_icon/` | `ind-` | 31 | Indicators |
| `UI_icon/` | *(none)* | 113 | Access, Search, Interest & Social, Hotel Amenities, Hotel Room Amenities, Pool Amenities |
| `Marketing_icon/` | *(none)* | 19 | Réassurance gradient (marketing-specific) |

**To find an SVG file from this catalog:**
- Navigation icons → `UI_icon/nav-{name}.svg` (e.g., "Check" → `nav-check.svg`)
- Indicator icons → `UI_icon/ind-{name}.svg` (e.g., "Star" → `ind-star.svg`)
- Amenity icons → `UI_icon/{name}.svg` (e.g., "Wifi" → `wifi.svg`)
- Marketing/Réassurance → `Marketing_icon/{name}.svg` (e.g., "Business" → `business.svg`)

### Sizing & Style:
- **Sizing**: 4 sizes available (S, M, L, XL). Default is M (24x24px).
- **Bold**: A bold variant exists for emphasis contexts.
- All icons are designed on a consistent grid for visual harmony.

## Export Instructions

To get individual icon PNGs for use in presentations:

1. Open the Design System file in Figma
2. Navigate to the Icons page (node `40:927`)
3. Select the icons you need (or select all in a category)
4. Right panel → Export → Choose format:
   - **SVG** for maximum quality and scalability
   - **PNG @2x** for direct use in PPTX (recommended: 128x128 or 256x256)
5. Export with transparent background
6. Place exported files in `assets/icons/` organized by category

### Batch export tip:
Select a category frame → right-click → "Select all children" → add Export settings → Export all at once.
